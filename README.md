## Описание проекта
Автоматизация тестирования формы заказа доставки карты с функцией перепланирования даты встречи.

## Использованные технологии и паттерны
* **Java 11**
* **Selenide** (UI-тесты)
* **Faker** (генерация случайных данных)
* **Lombok** (автогенерация кода)
* **Data Generator** (паттерн для подготовки тестовых данных)

## Инструкция по запуску
1. Склонируйте репозиторий.
2. Убедитесь, что файл `app-replan-delivery.jar` находится в папке `artifacts`.
3. Запустите приложение: `java -jar artifacts/app-replan-delivery.jar`.
4. Запустите тесты: `./gradlew test`.
